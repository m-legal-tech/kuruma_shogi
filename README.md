# 🚗 くるま将棋 🚒

子ども向けの車をテーマにした将棋風ゲームアプリ（PWA対応）

![くるま将棋](https://img.shields.io/badge/PWA-Ready-success)
![License](https://img.shields.io/badge/license-MIT-blue)

## 🎮 概要

「くるま将棋」は、5×5の盤面で車両を駒として使う、子ども向けの簡易将棋ゲームです。
将棋のルールをシンプルにし、親しみやすい車両キャラクターを使うことで、
お子様でも楽しく戦略ゲームに親しむことができます。

## ✨ 特徴

### 🎯 ゲームの特徴
- **5×5の盤面**: コンパクトで分かりやすい
- **5種類の車両駒**: 消防車・パトカー・救急車・ブルドーザー・スクールバス
- **持ち駒システム**: 取った駒を再配置できる将棋ルール
- **対面プレイ対応**: iPadを回転させて2人で対戦可能
- **バウンスアニメーション**: 自分のターンの駒が弾んで分かりやすい

### 📱 PWA機能
- **ホーム画面に追加可能**: ネイティブアプリのように使える
- **オフライン対応**: 一度読み込めばネット不要
- **高速起動**: キャッシュで瞬時に起動
- **全画面表示**: ブラウザUIなしの没入体験

### 🎨 デザイン
- **子ども向けUI**: カラフルで親しみやすい
- **レスポンシブ対応**: iPad/スマホで最適表示
- **音声ON/OFF**: 効果音の切り替え可能

## 🎲 遊び方

### 駒の種類と動き

| 車両 | 駒名 | 動き |
|------|------|------|
| 🚒 消防車 | 王将 | 全方向に1マス（取られたら負け）|
| 🚓 パトカー | 角 | 斜め方向に何マスでも |
| 🚑 救急車 | 飛車 | 縦横方向に何マスでも |
| 🚜 ブルドーザー | 金 | 前方と横に1マス |
| 🚌 スクールバス | 銀 | 斜め前と真前に1マス |

### 基本ルール

1. **青チーム（下側）と赤チーム（上側）**で対戦
2. **交互にターン**を進める
3. **相手の消防車を取ったら勝ち**
4. **取った駒は再配置可能**（持ち駒システム）
5. **自分のターンの駒が弾む**ので分かりやすい

## 📦 インストール

### iPadでPWA化

1. Safariで[デモURL](https://m-legal-tech.github.io/kuruma_shogi/kuruma-shogi.html)を開く
2. 画面下部の**共有ボタン** 📤 をタップ
3. **「ホーム画面に追加」**を選択
4. **「追加」**をタップ

→ ホーム画面に消防車アイコンが追加されます！

### Androidでインストール

1. Chromeで[デモURL](https://m-legal-tech.github.io/kuruma_shogi/kuruma-shogi.html)を開く
2. 右上メニュー ⋮ → **「ホーム画面に追加」**
3. または自動表示される**「インストール」**バナーをタップ

## 🛠️ 技術スタック

- **HTML5** - セマンティックマークアップ
- **CSS3** - フレックスボックス、グリッドレイアウト、アニメーション
- **Vanilla JavaScript** - フレームワーク不使用
- **Web Audio API** - 効果音生成
- **Service Worker** - オフライン対応、PWA化
- **PWA Manifest** - アプリ化設定

## 📂 プロジェクト構成

```
kuruma_shogi/
├── kuruma-shogi.html              # メインHTMLファイル
├── favicon.ico                    # ファビコン
├── public/                        # 静的アセット
│   ├── icon-192.png              # PWAアイコン 192×192
│   ├── icon-512.png              # PWAアイコン 512×512
│   ├── apple-touch-icon.png      # iOSアイコン 180×180

├── README.md                      # このファイル
└── DEPLOY-README.md              # デプロイ手順書
```

```bash

open http://localhost:3000/kuruma-shogi.html
```
